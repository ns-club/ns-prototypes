# 订单转发票信息提示弹窗(提示式)

OMS 订单批量转发票时的「提示式」校验弹窗原型。

## 背景

订单批量转发票提交时,部分 CO 单因**缺少仓库 / 仓库信息不一致**无法转发票。
原交互为**阻断式**(有错即拦截,整批无法提交);本原型改为**提示式**:
告知错误规模,允许用户跳过错误、仅提交正确数据,把决策权交给用户。

## 功能点

- 提示文案:本次提交 102 单,正确 70 单,信息不全 32 单(数字高亮)
- 「明细」可折叠展开,表格列:`CO 号 ｜ 错误原因 ｜ 结果`(表头吸顶、可滚动)
- 明细工具栏:**一键复制错误单号**(逗号分隔,便于粘贴批量搜索)、**导出清单**(CSV,带 BOM)
- 主操作:`跳过错误,仅提交正确数据(70)`;次操作:`取消`
- 头部、标题、按钮文案与配色对齐评审截图

## 使用

纯静态、无依赖。直接用浏览器打开 `index.html` 即可交互。
